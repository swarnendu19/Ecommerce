## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/yourproject.git
    ```

2. Navigate to the project directory:
    ```bash
    cd Backend
    ```

3. Install the dependencies and Run:
    ```bash
    docker run -it -p 3000:3000 swarnendu19/ecommerce-app
    ```
